# My config

My custom NVF configuration for nvim

## Want to try it out

If you have the Nix package manager or are using NixOS with flakes 
just run
```
nix run https://github.com/banana3216-2025/nvf-config
```
in your favorite
terminal

### features

- Catppuccin Mocha theming
- Dashboard
- Telescope(fuzzy finder)
- LSPs
- TODO highlighting
- Feeding the dog
